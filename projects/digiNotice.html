DigiNotice
This project is based on realworld problem that i faced during college days. So to get in touch with every notice and be alert for things happening in college we need a comman platform to provide for all the students and faculty members.
This project is worked in college and used by faculty members. Students are satisfied with the new digitaliztion of notice boards.

Technology:
WEB PORTAL: developed in php,html,css,bootstrap
Android App: Honeycomb to lollypop supported
Google cloud messaging servivce: notification service.
QR-code : QR-code api using ZXING android api.
